# dashboard-trafego-pago-powerbi
Dashboard de performance em Power BI para monitoramento de campanhas de mídia paga (Meta Ads). Contempla funil de conversão, métricas de eficiência (CTR, CPC, CPM, ROAS) e comparativo A/B entre campanhas. Análise resultou na pausa de campanha com custo por lead 67% maior e conversão 3x menor.
